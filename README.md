## Description
The user will be able to add expenses and deposits to their budget with or without a connection. When entering transactions offline, they should populate the total when brought back online.

## Installations

* npm init -y
* npm i express 
* npm i mongoose
* npm i morgan
* npm i nodemon
* npm i compression
* npm i require-stack
